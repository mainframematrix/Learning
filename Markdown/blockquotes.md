# Blockquotes

## Usage

### Simple Blockquotes

```markdown
> Add '>' infront of the paragraph
```

The above snippet renders as follows:

> Add '>' infront of the paragraph

### Multi-paragraph Blockquotes

```markdown
> Add '>' infront of the paragraph.
>
> Next Paragraph
```

The above snippet renders as follows:

> Add '>' infront of the paragraph
>
> Next Paragraph

| ✅ Do | ❌ Don't |
